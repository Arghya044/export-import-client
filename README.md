# Import Export Hub (Client)

Live Site: https://import-export-hub-arghya.netlify.app

## Features
- Modern SPA with React, Tailwind, and DaisyUI; responsive across devices.
- Firebase Authentication (Email/Password + Google) with persistent sessions.
- Browse all products and the latest 6 on the Home page, with search.
- Private routes: Product Details with “Import Now”, My Imports, Add Export, My Exports.
- Full CRUD: add/update/delete exports; import with quantity validation and real-time stock reduction.
- Dark/Light mode toggle with preference saved to localStorage.
- Toast-based feedback for all success/error actions (no browser alerts).



## Deployment
- Client: Netlify with Firebase Hosting.
- Server: Vercel.
